# Tennis Antella ASD

Sito statico per GitHub Pages.

## Pubblicazione
1. Carica tutti i file nella root del repository.
2. Imposta **Settings → Pages** su *Branch: main* e *Folder: /root*.
3. Attendi 1–2 minuti e visita `https://<tuo-username>.github.io/TennisAntella/`.

## Personalizzazioni
- Sostituisci le immagini in `assets/img/`.
- Aggiorna i documenti PDF in `/docs`.
- Modifica colori in `assets/css/style.css` (variabili CSS nella prima riga).
